## 8. 본 PoC 와의 정렬도 — 어필 포인트 매핑

| 채용 키워드 | 본 PoC 가 증명하는 부분 |
|---|---|
| **C# / ASP.NET Core 기반 대규모 서버** | `Server/` — Kestrel + ASP.NET Core 10 + MagicOnion StreamingHub |
| **네트워크 프로그래밍** | gRPC(HTTP/2) 위 양방향 RPC, `IMovementHub` 계약 설계 |
| **성능 튜닝** | Phase 2 — `ArrayPool` / `Span` 기반 Zero-Allocation 최적화, BenchmarkDotNet 으로 5,000명 기준 **3.8x 속도, 할당 82KB→0B** ([BENCHMARK.md](BENCHMARK.md)) |
| **운영/관측** | `/api/metrics` + Chart.js 시계열 + Heatmap 으로 GC / TPS / AOI 실시간 시각화 |
| **설계** | `Shared / Server / BotClients / Benchmarks` 4-프로젝트 구조, 멀티룸 그룹 추상화 |
| **부하 시뮬레이션** | `BotClients` — 단일 프로세스 수천 세션 헤드리스 부하기 |
